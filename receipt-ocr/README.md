# Receipt OCR

Project scaffold for receipt image OCR, structured for preprocessing, OCR, extraction, and confidence scoring.

## Structure

- `data/` - receipt images
- `outputs/` - JSON results
- `src/preprocess.py` - image preprocessing
- `src/ocr.py` - OCR execution
- `src/extract.py` - field extraction
- `src/confidence.py` - confidence scoring
- `src/main.py` - pipeline entrypoint
